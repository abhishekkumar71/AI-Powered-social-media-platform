# AI-Powered Social Media Platform

This project is a web application that allows users to generate AI content and post to multiple social media platforms (X/Twitter, Instagram, LinkedIn) from a single dashboard. Users can also schedule posts and automate their social presence.

## Tech Stack

- **Frontend:** Next.js, React, Material UI (MUI), Tailwind CSS
- **Backend:** Node.js, Next.js API routes
- **Authentication:** NextAuth.js with Google OAuth
- **Database:** Prisma ORM with PostgreSQL / MySQL (configurable)
- **Social Integrations:** X/Twitter API, Instagram API
- **Deployment:** Vercel or any cloud platform

## Features

- Generate AI content (captions, posts) instantly
- One-click posting to multiple social platforms
- Dashboard to manage and track your posts
- Secure authentication using Google OAuth
- Real-time notifications for post success/failure
